# King-Slime-PCB

 The project is a model of the King slime in Terraria. It also has the added function of vibrating and lighting up when the switch is clicked.
 
## Bill of meterials 
-  Resistors 6x
- 5mm LED 6x
 - mini motor disc
 - 6mm push button
 - CR2032 battery cell
